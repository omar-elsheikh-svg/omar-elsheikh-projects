# 🚀 Python Task & Skill Tracker (CSV-Based)

A robust Command-Line Interface (CLI) application designed to manage tasks and track skill progress. This project focuses on **Data Persistence** and **Clean Code** principles, built as part of my advanced Python training.

---

## 🛠️ Key Features
- **Data Persistence:** Uses CSV files to store data, ensuring your progress is saved even after closing the app.
- **Error Handling:** Implements `try-except` blocks to handle invalid inputs and missing files gracefully.
- **Interactive CLI:** A user-friendly menu system for seamless navigation (Add, Remove, View).
- **Clean Architecture:** Modular functions for better readability and maintainability.

---

## 💻 Tech Stack
- **Language:** Python 3.x
- **Storage:** CSV (Comma Separated Values)
- **Version Control:** Git & GitHub
